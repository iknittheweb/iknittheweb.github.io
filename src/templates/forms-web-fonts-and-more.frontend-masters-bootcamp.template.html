Forms, Web Fonts, and More

This project demonstrates the use of modern HTML form elements and web fonts, as covered in the Frontend Masters Bootcamp. It includes examples of various input types—text, email, phone, date, number, radio, checkbox, textarea, and select—each with best-practice markup and accessibility considerations. The project also demonstrates how to use web fonts from providers like Google Fonts and FontAwesome, and how to embed external content such as maps and videos using <iframe> and <embed> tags.
